ClawDeck Menu Bar App — wraps DeckController in a macOS menu bar interface.
 
Provides:
  - Start/Stop Stream Deck controller
  - Status indicator in menu bar
  - Settings window (local HTTP server + browser)
  - Session-map rescan
  - Install/update hooks
